WeCheat
====

模仿Wechat服务器行为，辅助微信公众平台开发。

Install
----
```
git clone https://github.com/xixilive/wecheat

bundle install
```

Setup and Purge data for mock
----
```
rake setup or purge
```

Launch App
----
```
rackup
```
